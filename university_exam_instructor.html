Define overfitting in machine learning and explain how it can be prevented.
Expected Answer: Overfitting occurs when a model learns the training data too well, including noise and random fluctuations, leading to poor generalization. It can be prevented through cross-validation, regularization, early stopping, and using more training data.

5.

Compare and contrast supervised and unsupervised learning. Provide examples of algorithms for each type and discuss when you would use each approach.
Expected Answer: Supervised learning uses labeled training data to learn a mapping function, examples include linear regression and random forest. Unsupervised learning finds patterns in unlabeled data, examples include k-means and hierarchical clustering. Use supervised when you have target variables, unsupervised for exploratory data analysis and pattern discovery.
